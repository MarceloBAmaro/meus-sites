1-formatação

    1.1-hieranquia dos titulos

    os titulos tem uma hieranquia que se resume a o numero que vem depois de H
    ex: <H1></H1>
    o número pode ir até 6, caso o número for maior de 6 o titulo NÃO sera contado porém as palavras serão escritas
    ex: <H10>olá, mundo</H10>
    o "olá, mundo" seria escrito mas sem formatação de titulo nenhum

    1.2-principais formatações

    há 14 principais formatações

    negrito/destaque
    é escrito com o comando <STRONG>, deixa as palavras selecionadas com uma espessura mais forte
    ex: <STRONG>esta é uma frase em negrito/destaque</STRONG>
    formaria a frase: esta é uma frase em negrito/destaque
    italico/destaque
    é escrito com o comando <EM>, deixa as palavras selecionadas com um aspecto curvado
    ex: <EM>esta é uma frase em italico/destaque</EM>
    formaria a frase: esta é uma frase em italico/destaque
    texto marcado
    é escrito com <MARK>, deixa as palavras escolhidas marcadas(obvio)
    ex: <MARK>esta é uma frase marcada</STRONG>
    formaria a frase: esta é uma frase marcada.
    a cor pode ser mudada com o comando <STYLE> dentro de MARK
    ex: <MARK STYLE="background-color: blue">frase marcada com a cor azul</MARK>(sendo "azul" a cor, que fica a sua escolha)
    agora a frase é: frase marcada com a cor azul
    ou pode ser mudada com o comando na cabeça(<head>)
    ex: <head>
    <STYLE>
    MARK{backgraund-color="blue"}
    </STYLE>
    </head>
    neste exemplo a cor seria usada de modo geral em todos os MARKS se já NÃO estiver definido a cor(ou qualquer outra customização) dentro do MARK
    texto pequeno
    é escrito com <SMALL>, deixa as palavras selecionadas menores
    ex: <SMALL>esta é uma frase com letras pequenas</SMALL>
    formaria a frase: esta é uma frase com letras pequenas
    texto excluido
    é escrito com <DEL>, faz com que as palavras elecionadas sejam mostradas porém não consideradas
    ex: <DEL>esta é uma frase excluida</DEL>
    formaria a frase: esta é uma frase excluida
    texto inserido
    é escrito com <INS>, faz com que as palavras selecionadas ganhem êmfase para mostrar que foram inseridas
    ex: <INS>esta é uma frase com êmfase/inserida</INS>
    formaria a frase: esta é uma frase com êmfase/inserida
    texto sobrescrito
    é escrito com <SUP>, é usada para escrever coisas do tipo X2
    ex de equação: X<SUP>3</SUP> + 5
    formaria a equação: X3 + 5
    texto subscrito
    é escrito com <SUB>, é usado para escrever coisas do tipo H2O
    ex: a molecula da água é H<SUB>2</SUB>O
    formaria a frase: a molecula da água é H2O
    codigo-fonte
    é esrcrito com <CODE>, deixa as letras com o mesmo tamanho
    ex: libidinoso -> libidinoso
    codigo-indentação
    é escrita com o codigo <PRE>, mantem a indentação(arrumação) que estava no codigo
    ex: <PRE>este é um codigo que
    mantem a indentação
    original</PRE> formaria a frase:
        este é um cogigo que
    mantem indentação
    original
    a letra esta menor porque normalmente se utiliza para escrever codigos de programação
    citação
    é escrito com <Q>, serve para mostrar para o sistema que as palavras escritas são uma citação
    ex: como já dizia socrates:<Q>só sei que nada sei</Q>
    formaria a frase: como já dizia socrates:só sei que nada sei
    citações em bloco
    é escrita com <CITE>, se usa para escrever uma ciação com mais de um paragrafo e pode conter o link do site de onde você tirou essa citação
    ex: freud: <CiTE>Antes de diagnosticar a si mesmo comdepressão ou baixa-autoestima, primeiro tenha certeza de que você não está, de fato, cercado por IDIOTAS.</CITE>
    formaria a frase: freud: Antes de diagnosticar a si mesmo com depressão ou baixa-autoestima, primeiro tenha certaza de que você não está, de fato, cercado por IDIOTAS
    abreviações
    é escrita com <ABBR>, serve para que quando o usuario passar o mouse em cima da abreviação mostre o nome inteiro
    ex: <ABBR TITLE="HyperText Markup Language">html</ABBR>
    formaria a frase: html
    (coloque o mouse em cima)
    texto invertido
    é escrito com <BDO>, faz com que as palavras selecionadas sejam escritas invertidas
    ex: <BDO DIR="rtl"(right to left(direita para esquerda))>esse é um texto invertido</BDO>
    formaria a frase: esse é um texto invertido
    o rtl pode ser mudado para ltr(left to right(esquerda para direita))

1.3-simbolos especiais

  1. marca registrada ®
    é representada por &reg;
  2. copyright ©
    é representada por &copy;
  3. fração
    a fração é representada pelo
  4. grau ˚
    é representado por &ring;
  5. delta Δ
    é representado por &Delta;
    (o "D" tem que estar em maiusculo)
  6. pi π
    é representado por &pi;
  7. alfa, beta e gama α, β e γ
    alfa α, é representado por &alpha;
    beta β, é rapresentado por &beta;
    gama γ, é representado por &gamma;
  8. reticências …
    é representado por &hellip;
  9. infinito ∞
    é representado por &infin;
  10. numero №
    é representado por &numero;
  11. comercio ™
    é representado por &trade;
  12. moedas
    euro € é repreentado por &euro;
    dollar $ é representado por &dollar;
    real ℜ$ é representado por &real;+&dollar;
    yen ¥ é representado por &yen;
  13. blocos █ ▒ ░ ▓
    █ é representado por &block;
    ▒ é representado por &blk12;
    ░ é representado por &blk14;
    ▓ é representado por &blk34;
  14. sexo ♀ ♂
    ♀ é representado por &female;
    ♂ é representado por &male;
  15. flechas ↑ ↓ ← →
    as flechas são representas por &#arr;
    sendo # o lado que pode ser L(esquerda) R(direita) U(cima) D(baixo)

    16. 1.4-listas

    1.4.1-listas ordenadas

    são representas por <OL>
    ex: <OL>
    <LI>euro</LI>
    <LI>dollar</LI>
    <LI>real</LI>
    </OL>
    formaria:
    1. euro
    2. dollar
    3. real
    a representação dos numeros pode ser mudada para numeros romanos maisculos ou minusculos ou letras maiusculas ou minusculas:
      1. letra maiuscula
        é escrita com <OL TYPE="A">
      1. letra minuscula
        é escrita com <OL TYPE="a">
      1. romana minuscula
        é escrita com <OL TYPE="i">
      1. romana maiuscula
        é escrita com <OL TYPE="I">
    podem ser inicidas do ponto onde você quiser com o comando <OL START="#">
    sendo # o ponto de inicio(a escala de conta só aceita numeros)
    o tipo de lista e o inicio da lista podem ser combinados

    1.4.2-listas desordenadas

    é escrita com <UL>
    ex: <UL>
    <LI>pão</LI>
    <LI>queijo</LI>
    </ul>
    formaria:
    • pão
    • queijo

    o tipo de lista pode ser mudada para disco ou circulo vazado ou quadrado
    1. disco
      é escrito com <UL TYPE="DISC"
    2. circulo vazado
      é escrito com <UL TYPE="CIRCLE">
    3. quadrado
      é escrito com <UL TYPE="SQUARE"

    1.4.3-listas de definição

    uma lista de definição é escrita com <DT>(termo, palavra-chave) e <DD>(definição da palavra chave) dento de <DL>(corpo da lista)
    ex: <DL><DT>html</DT><DD>é uma linguagem de marcação e usa-se para criar cites</DD><DT>
    formaria:
    html
    é uma linguagem de marcação e usa-se para criar cites

2-importação de imagem, video e audio

    2.1-importação de imagem

    uma imagem pode ser importada com ocomando <IMG>
    ex: <IMG SRC="onde procurar" ALT="contra medida se caso a imagem não for carregada">
    a sua altura e largura podem ser mudadas com os comandos width(largura) e height(altura)
    ex: <IMG WIDTH="largura" HEIGHT="altura" SRC="" ALT="">
    as imagems podem ser importadas do computador ou de um link

    2.1.1-imagem dinamica

    a diferenciação de imagem por midea se constroi com o comando <SOURCE> dentro de <PICTURE>
    ex: <PICTURE>
    <SOURCE SRCEST="onde procurar" MEDIA="(@-!: (numero em px) REL="image/">
    <IMG SRC="onde procurar">
    </PICTURE>

    2.3-importação de icone de aba

    é escrito com <LINK:FAVICOM REL="shortcut icon" href="onde procurar" TYPE="IMAGE/x-icon"> sendo @ max(maximo) ou min(minimo) e ! sendo width(largura) e heigth(altura) (é necessario evidenciar o tipo de ação com o camdo rel="tipo")

    2.2-importação de video

    os videos podem ser importados com o comando <VIDEO>
    ex: <VIDEO SRC="onde procurar" CONTROLS></VIDEO>
    porem há outro jeito melhor de importar videos que é:
    <VIDEO CONTROLS>
    <SOURCE SRC="onde procurar" TYPE="o tipo de arquivo">
    </VIDEO>
    essa forma de importação é um efeito em cascata onde o comando tenta acionar o primeiro primeiro que o segundo e assim por diante
    o comando pode conter a largura(width="largura") e/ou a altura(heigth="altura")
    pode conter a tag LOOP
    tambem pode conter a tag <POSTER> que faz uma foto ser colocada como tela de descanso do video.

3-importação de objetos exteriores

    3.1-link

    é escrito com <A> ex: <A HREF="https://www.youtube.com/watch?v=ZfWv96cxp9I">este é um link</A>
    formaria:este é um link
    pode conter a tag <TITLE> para que quando o usuario coloque o mouse em cima do link apareça um titulo
    ex: <A HREF="https://www.youtube.com/watch?v=ZfWv96cxp9I" TITLE="este é o titulo">este é um link com titulo</A>
    formaria: este é um link com titulo
    (coloque o mouse em cima do link)
    é recomendavel colocar o <REL="#"> para informar se o link é externo ou interno
    sendo # external(externo) ou next(proximo) ou prev(anterior) ou nofollow(site que não tem seu aval).
    ex de link interno: segunda pagina
    para abrir o link em outra pagina basta colocar o comando <TARGET="_blank"> no link para abrir na mesma pagina usa-se o comando <TARGET="_self"> no link

    3.1.1-dowload por link

    são escritos com <A HREF="onde procurar" DOWLOAD="o que é para baixar" TYPE="o tipo de arquivo que é para baixar"
    o tipo de arquivo pode ser encontrado no site da iana
    ex: <A HREF="PDF/TESTE.pdf" DOWLOAD="PDF/TESTE.pdf" TYPE="application/pdf" >clique aqui para baixar um arquivo pdf</A>
    formaria: clique aqui para baixar um arquivo pdf
